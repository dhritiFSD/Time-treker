# Teen Screen Guard

Teen Screen Guard is a responsive website that helps teenagers track and manage their daily screen time while promoting healthy digital habits.

---

## Features

- Screen Time Tracker
- Start, Stop, and Reset Timer
- Responsive Navigation Bar
- Healthy Digital Habit Tips
- Modern UI Design
- Mobile Friendly Website

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
